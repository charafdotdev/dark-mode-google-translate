# Contributing to Dark Mode for Google Translate™

Thank you for helping make this extension better!

## How to contribute

1. Fork the repository
2. Create a new branch (`git checkout -b your-feature`)
3. Make your changes
4. Test thoroughly on https://translate.google.com
5. Commit with a clear message
6. Push and open a Pull Request

## We especially welcome

- Fixes for new Google Translate layout changes
- Better contrast or color tweaks
- Performance improvements
- Updated screenshots

All PRs are reviewed quickly. Feel free to open one – even tiny fixes are appreciated!

Happy coding!
